
## Actualización del manual

Se agregó esta sección desde la rama desarrollo para demostrar el flujo colaborativo en GitHub.

## Procedimiento

1. Se creó una rama llamada desarrollo.
2. Se modificó el archivo docs/manual.md.
3. Se guardaron los cambios con un commit.
4. Se envió la rama al repositorio remoto.
5. Posteriormente se creará un Pull Request para integrar los cambios a la rama principal.
