
# PROYECTO_IRLANDA_MENDIETA

Este proyecto fue creado como parte de la práctica integral de Git y GitHub.

## Descripción

El objetivo del proyecto es practicar la administración de repositorios locales y remotos, utilizando comandos de Git y herramientas de GitHub.

## Contenido del repositorio

- Carpeta src: contiene archivos de ejemplo del proyecto.
- Carpeta docs: contiene documentación del proyecto.
- Archivo README.md: describe el propósito general del repositorio.

## Herramientas utilizadas

- Git
- GitHub
- Terminal de macOS
- Visual Studio Code
